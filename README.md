# Installation

```bash
$ git clone git@github.com:yukpiz/nicovideo-dl.git
$ cd nicovideo-dl

$ brew install chromedriver
$ chromedriver --version
ChromeDriver 2.34.522932 (4140ab217e1ca1bec0c4b4d1b148f3361eb3a03e)
```

# Usage

```
$ ./nicovideo-dl -l [Playlist URL]
```
